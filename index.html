Lab Registry / reagents
R-2024-0001

GitHub Pages setup

1

Create a GitHub repository

Go to github.com/new, create a new public repository (e.g. lab-registry). You only need to do this once.

2

Upload this file

Click Add file → Upload files in your repo and upload lab-registry.html. Rename it to index.html so it loads automatically.

3

Enable GitHub Pages

Go to your repo → Settings → Pages. Under Source, select main branch and / (root). Click Save. Your URL will be:

https://YOUR-USERNAME.github.io/lab-registry/
4

Workflow for adding entries

Add entries here on your computer → click Export HTML below → this downloads an updated copy of the file with your data baked in → upload/replace on GitHub → the page updates in ~1 minute. The QR codes you print will always point to the live URL.

5

QR code URL pattern

When you print a label, the QR encodes a URL like:

https://YOUR-USERNAME.github.io/lab-registry/#R-2024-0001

Scanning opens the registry and jumps straight to that entry. Update the base URL below after setting up.

(no trailing /)
