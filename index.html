Setup & GitHub sync

1

Create a GitHub repository

Go to github.com/new. Name it (e.g. specim), set it to Public, and click Create repository. Upload this index.html file to the main branch.

2

Enable GitHub Pages

Go to Settings → Pages, set source to Deploy from branch → main / root. After ~1 min your registry is live.

https://schmidtdav.github.io/specim/
3

Create a data branch with db.json

Branch dropdown → type data → Create. Switch to it, add db.json with content [].

4

Create a Personal Access Token

GitHub → Settings → Developer settings → Fine-grained tokens. Scope to this repo, Contents: Read and Write.

5

Paste your token below

Changes are saved to db.json in your GitHub repo. Token stored in browser only.
